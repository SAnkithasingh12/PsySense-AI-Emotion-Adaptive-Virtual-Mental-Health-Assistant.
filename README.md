# PsySense-AI-Emotion-Adaptive-Virtual-Mental-Health-Assistant.
Project Description
PsySense AI is an emotion-adaptive virtual mental health assistant designed to support users by analyzing their emotional state through text input and providing suitable responses.
This project aims to improve mental well-being by offering interactive chatbot conversations, emotion analytics, and chat history tracking.

🎯 Objectives
To detect user emotions from text input
To provide supportive and adaptive chatbot responses
To maintain conversation history for future reference
To display emotion insights using analytics

⭐ Key Features
Emotion-based chatbot responses
User dashboard for quick overview
Analytics page to view emotion trends
History page to store previous conversations
Profile page to manage user details
Simple and user-friendly interface

📂 Project Modules
Dashboard Page – Displays system overview and user activity summary
Chatbot Page – User interacts with the AI chatbot for mental health support
Analytics Page – Shows emotional insights and analysis reports
History Page – Stores and displays previous chat conversations
Profile Page – Displays and manages user profile information

🛠️ Technologies Used
Python
Flask Framework
HTML
CSS
JavaScript
SQLite Database
NLP (Natural Language Processing)

📌 Output
The system detects the emotion from the user’s text and generates emotion-adaptive responses.
It also provides analytics and stores chat history for user tracking.
